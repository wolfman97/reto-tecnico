# Leasein Backend

API REST para gestión y validación de equipos tecnológicos.

## Tecnologías
- Laravel 10
- PostgreSQL

## Endpoints
- GET /api/equipos
- POST /api/validar-equipos

## Instalación
1. Clonar repositorio
2. Configurar .env
3. composer install
4. php artisan migrate --seed
5. php artisan serve

# Leasein Frontend

Aplicación Vue 3 para visualización y filtrado de equipos.

## Tecnologías
- Vue 3
- PrimeVue
- Axios

## Instalación
1. npm install
2. npm run dev
